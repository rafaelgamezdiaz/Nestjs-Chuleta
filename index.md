## Nestjs-Chuleta



### Instalación

Instalar de forma global

```
 npm i –g @nestjs/cli
```


### Crear un proyecto

```
 nest new project_name
``` 


### Iniciar el proyecto en modo desarrollador

```
 npm run start:dev
```


### Crear un Modulo

```
 nest g module tasks
```



### Crear un Controlador


```
 nest g controller tasks
```
